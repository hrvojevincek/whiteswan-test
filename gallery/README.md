# Gallery preparation (manual enrollment)

Place **3–10** face crops per character under this directory.

## Directory structure (slugs)

Use these folder names exactly:

- `harry` → `Harry Potter`
- `ron` → `Ron Weasley`
- `hermione` → `Hermione Granger`
- `mcgonagall` → `Prof. McGonagall`
- `snape` → `Prof. Severus Snape`

## What to include in each crop

- Tight, face-centered crops (include the full face, but avoid lots of background)
- Different angles help, especially for `McGonagall` vs `Snape`
- Avoid very blurry or heavily occluded images

Any image format supported by OpenCV/DeepFace is fine (`.jpg`, `.png`, etc.).

## Provenance note

If you want, add a short note per character (e.g. approximate timestamps) so you can explain your gallery choice in your submission.
